# Setup

Después de que se montó la organización y cluster, toca conectarse a este último.
Para hacerlo, hay diferentes métodos.

## Connecting with MongoDB for VS Code

1. Instalar la extensión.
2. Abrir la command pallete y buscar `MongoDB:Connect` y darle click a `Connect with Connection String.`.
3. Copiar el string de conexión.

> **Ejemplo de string de conexión:**
> ```bash 
> mongodb+srv://quesadx:<db_password>@cineplexcluster.xeej5kl.mongodb.net/
> ````
>
