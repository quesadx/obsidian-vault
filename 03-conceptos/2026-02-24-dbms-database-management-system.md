# DBMS (Database Management System)

Un **Database Management System** es un software que permite crear, administrar y controlar bases de datos.

## Definición Formal

> Un DBMS es una **colección de datos interrelacionados** con programas para acceder a ellos.

## Características Principales

### Abstracción

La abstracción permite a un usuario utilizar un dispositivo complejo sin tener que conocer los detalles internos de cómo funciona.

- El usuario interact con una interfaz simple
- El DBMS maneja la complejidad interna

### Ejemplos de DBMS

- **MySQL**
- **PostgreSQL**
- **Oracle Database**
- **Microsoft SQL Server**
- **MongoDB** (NoSQL)
- **Redis** (NoSQL)

## Funciones del DBMS

1. **Almacenamiento y recuperación de datos**
2. **Cumplimiento de integridad**
3. **Control de concurrencia**
4. **Recuperación ante fallos**
5. **Control de acceso y seguridad**
6. **Mantenimiento de índices**
