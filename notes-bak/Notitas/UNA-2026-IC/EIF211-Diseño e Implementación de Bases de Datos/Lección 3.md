## Diseño y Evolución de Base de Datos 
### Las 3 eras del dato
1. RDMBS: Para sistemas transaccionales.
2. Data warehouse: Para aplicaciones analíticas.
3. NoSQL: Para cualquier contenido. 
### RDBMS
Permite consultas rápidas para grandes volúmenes estructurados, pero no es optimizado para transacciones masivas en tiempo real ni actualizaciones frecuentes. 
- Almacenamiento en filas y columnas.
- Modelo de datos pre-definido (esquema rigido).
- Lenguaje estandarizado (SQL).
- Relaciones mediante joins.
