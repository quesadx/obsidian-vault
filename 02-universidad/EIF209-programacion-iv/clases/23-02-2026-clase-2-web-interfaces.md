# 2026-02-23

## Desarrollo de interfaces web
Corresponde a un contrato visual e interactivo.

Combina tres pilares:
* Estructura (HTML)
* Presentación (CSS)
* Comportamiento (JS)

Impacta directamente en la conversión, productividad, riesgo legal y costo de soporte de la aplicación.

## HTML
Hypertext Markup Language:
* No programa
* No da estilos
* No maneja lógica

Etiquetas fundamentales:
* header
* nav
* main
* section
* article
* aside
* footer


## CSS
Se aplica plantilla
Colores
Diseño final

CSS distinto a la hora de imprimir
