## SOLUCIONARIO DETALLADO

### SECCIÓN 1

**Ejercicio 1**  
*Procedimiento y justificación:* Para ubicar un punto `(a, b)` en el plano cartesiano, se traza una recta vertical `x = a` y una recta horizontal `y = b`. La intersección de ambas corresponde al punto solicitado. Por definición, los puntos sobre el eje x tienen ordenada cero, los puntos sobre el eje y tienen abscisa cero, y el origen es `(0, 0)`.  
*Respuesta:* `A(−2, 3)` se ubica en el segundo cuadrante. `B(0, −1)` pertenece al eje y. `C(4, 0)` pertenece al eje x. `D(−1, −4)` se ubica en el tercer cuadrante. `E(0, 0)` corresponde al origen del sistema.

**Ejercicio 2**  
*Procedimiento y justificación:*  
a) La pendiente `m` mide la razón de cambio constante entre las ordenadas y las abscisas. Se calcula con `m = (y₂ − y₁) / (x₂ − x₁)`.  
b) Una vez obtenida `m`, se utiliza la forma punto-pendiente `y − y₁ = m(x − x₁)` y se despeja `y` para obtener `y = mx + b`. Esta forma es útil porque muestra directamente la inclinación y el corte con el eje y.  
c) Para la forma general, se trasladan todos los términos a un lado hasta obtener `Ax + By + C = 0`.  
*Respuesta:* a) `m = (4 − (−2)) / (4 − 1) = 6 / 3 = 2`.  
b) Usando `M(1, −2)`: `y − (−2) = 2(x − 1) → y + 2 = 2x − 2 → y = 2x − 4`.  
c) `2x − y − 4 = 0`.

**Ejercicio 3**  
*Procedimiento y justificación:*  
a) La intersección con el eje y ocurre cuando `x = 0`. La intersección con el eje x ocurre cuando `y = 0`. Se sustituyen estos valores en la ecuación original.  
b) Se despeja `y` para identificar la pendiente en la forma `y = mx + b`. Si `m > 0` la función es creciente, si `m < 0` es decreciente y si `m = 0` es constante.  
*Respuesta:* a) Eje y (`x=0`): `−2y + 6 = 0 → y = 3`. Punto: `(0, 3)`. Eje x (`y=0`): `3x + 6 = 0 → x = −2`. Punto: `(−2, 0)`.  
b) Despejando: `2y = 3x + 6 → y = (3/2)x + 3`. Como `m = 3/2 > 0`, la recta es **creciente**. Esto indica que al aumentar `x`, el valor de `y` también aumenta.

**Ejercicio 4**  
*Procedimiento y justificación:* En la ecuación `y = mx + b`, `m` representa la pendiente y `b` la ordenada al origen. Se ubica el punto `(0, b)` y, utilizando la pendiente, se calcula un segundo punto para trazar la recta.  
*Respuesta:* Pendiente `m = −1/2`, ordenada al origen `b = 3`. La recta corta al eje y en `(0, 3)`. Como la pendiente es negativa, por cada 2 unidades que se avanza a la derecha en `x`, se desciende 1 unidad en `y`. Un segundo punto es `(6, 0)`. La gráfica es una línea recta descendente.

### SECCIÓN 2

**Ejercicio 5**  
*Procedimiento y justificación:* Dos rectas son paralelas si y solo si sus pendientes son iguales (`m₁ = m₂`). Se despeja `y` en `l₁` para hallar su pendiente, luego se usa esa misma pendiente con el punto `P` para hallar la nueva ordenada al origen.  
*Respuesta:* `l₁: y = −2x + 5 → m = −2`. Como `l₂` es paralela, `m₂ = −2`. Sustituyendo en `y = mx + b` con `P(1, 1)`: `1 = −2(1) + b → b = 3`. Ecuación de `l₂`: `y = −2x + 3`, o en forma general: `2x + y − 3 = 0`.

**Ejercicio 6**  
*Procedimiento y justificación:* Dos rectas son perpendiculares si el producto de sus pendientes es `−1` (`m₁ · m₂ = −1`), lo que equivale a `m₂ = −1/m₁`. Se despeja `y` en `l₄` para obtener `m₁`, se calcula `m₂`. Como pasa por el origen `(0, 0)`, su ordenada al origen es `b = 0`.  
*Respuesta:* `l₄: 3y = x + 4 → y = (1/3)x + 4/3 → m₁ = 1/3`. Pendiente perpendicular: `m₂ = −1/(1/3) = −3`. Pasa por `(0, 0) → b = 0`. Ecuación de `l₃`: `y = −3x`, o `3x + y = 0`.

**Ejercicio 7**  
*Procedimiento y justificación:* Los tres métodos algebraicos buscan eliminar una variable para resolver la otra, basándose en el principio de equivalencia de ecuaciones. El punto de intersección satisface ambas ecuaciones simultáneamente.  
*Respuesta:* Punto de intersección: `(2, 3)`.  
- **Sustitución:** Se despeja `y` en la primera ecuación (`y = 4x − 5`) y se sustituye en la segunda. Esto reduce el sistema a una ecuación con una sola variable. `2x + 3(4x − 5) = 13 → 14x = 28 → x = 2`. Sustituyendo en `y = 4x − 5` se obtiene `y = 3`.  
- **Eliminación:** Se multiplica la primera ecuación por 3 para igualar coeficientes de `y`: `12x − 3y = 15`. Se suma con la segunda (`2x + 3y = 13`). La suma elimina `y`: `14x = 28 → x = 2`. Se sustituye en cualquiera de las ecuaciones originales para hallar `y = 3`.  
- **Igualación:** Se despeja `y` en ambas: `y = 4x − 5` y `y = (−2x + 13)/3`. Como ambas expresan la misma `y`, se igualan: `4x − 5 = (−2x + 13)/3`. Se multiplica por 3 y se resuelve: `12x − 15 = −2x + 13 → 14x = 28 → x = 2`. Luego `y = 3`.

### SECCIÓN 3

**Ejercicio 8**  
*Procedimiento y justificación:* Se evalúa el criterio `f(x)` en cada elemento del dominio para hallar las imágenes. El ámbito es el conjunto formado exclusivamente por esas imágenes. Para hallar la preimagen de un valor `k`, se iguala `f(x) = k` y se despeja `x`, verificando que el resultado pertenezca al dominio.  
*Respuesta:* a) Imágenes: `f(−2) = −13`, `f(0) = −7`, `f(3) = 2`, `f(5) = 8`. Conjunto de imágenes: `{−13, −7, 2, 8}`.  
b) Ámbito = `{−13, −7, 2, 8}`.  
c) Preimagen de 2: `3x − 7 = 2 → 3x = 9 → x = 3`. (Verifica: `3 ∈ D_f`).  
d) `f(−1) = 3(−1) − 7 = −10`.

**Ejercicio 9**  
*Procedimiento y justificación:* Se aplica el mismo principio de evaluación y despeje. Se analiza el grado del polinomio para determinar si es lineal.  
*Respuesta:* a) `g(3) = 3² + 1 = 10`.  
b) `x² + 1 = 10 → x² = 9 → x = 3` o `x = −3`.  
c) **No es una función lineal.** Una función lineal debe tener la forma `f(x) = mx + b`, es decir, ser un polinomio de primer grado. En este caso, el término `x²` indica que es una función cuadrática (grado 2), cuya gráfica es una parábola, no una recta.

**Ejercicio 10**  
*Procedimiento y justificación:* Se aplica el criterio a cada elemento de `A`. Los elementos de `A` forman el dominio y el conjunto de preimágenes. Los resultados obtenidos forman el conjunto de imágenes. El ámbito es idéntico al conjunto de imágenes. El codominio es el conjunto `B` dado.  
*Respuesta:* a) Dominio: `{1, 2, 3}`. Codominio: `{2, 4, 6, 8, 10}`. Preimágenes: `{1, 2, 3}`. Imágenes: `{2, 4, 6}`. Ámbito: `{2, 4, 6}`.  
b) La gráfica consiste en tres puntos discretos en el plano: `(1, 2)`, `(2, 4)` y `(3, 6)`. No se unen con una línea continua porque el dominio es un conjunto finito discreto.

### SECCIÓN 4

**Ejercicio 11**  
*Procedimiento y justificación:* Se identifica el tipo de función. Las funciones polinomiales no presentan restricciones algebraicas en los números reales, por lo que están definidas para todo valor real.  
*Respuesta:* `D_f = ℝ`, o en notación de intervalos: `(−∞, ∞)`.

**Ejercicio 12**  
*Procedimiento y justificación:* Es una función racional. El denominador no puede ser cero, ya que la división por cero no está definida. Se iguala el denominador a cero, se resuelve la ecuación y se excluyen esas soluciones del conjunto de los números reales.  
*Respuesta:* `x² − 9 = 0 → (x − 3)(x + 3) = 0 → x = 3, x = −3`. Dominio: `ℝ − {−3, 3}`, o `(−∞, −3) ∪ (−3, 3) ∪ (3, ∞)`.

**Ejercicio 13**  
*Procedimiento y justificación:* Es una función radical con índice par (raíz cuadrada). En los números reales, el radicando debe ser mayor o igual a cero para que el resultado sea un número real. Se plantea la inecuación y se resuelve.  
*Respuesta:* `2x − 6 ≥ 0 → 2x ≥ 6 → x ≥ 3`. Dominio: `[3, ∞)`.

**Ejercicio 14**  
*Procedimiento y justificación:* La raíz cúbica (índice impar) está definida para cualquier número real, pero al estar en el denominador, no puede ser cero. Se iguala el radicando a cero y se excluye ese valor.  
*Respuesta:* `∛(x − 5) ≠ 0 → x − 5 ≠ 0 → x ≠ 5`. Dominio: `ℝ − {5}`, o `(−∞, 5) ∪ (5, ∞)`.

**Ejercicio 15**  
*Procedimiento y justificación:* Se combinan dos restricciones: la raíz cuadrada (radicando `≥ 0`) y el denominador racional (denominador `≠ 0`). Se resuelven ambas condiciones y se halla la intersección de los conjuntos resultantes.  
*Respuesta:* Condición 1 (radical): `x + 1 ≥ 0 → x ≥ −1`. Condición 2 (denominador): `x − 3 ≠ 0 → x ≠ 3`. Intersección: `x ≥ −1` y `x ≠ 3`. Dominio: `[−1, 3) ∪ (3, ∞)`.

### SECCIÓN 5

**Ejercicio 16**  
*Procedimiento y justificación:*  
a) Se usa la fórmula de la pendiente con los dos puntos dados para hallar `m`. Luego se sustituye `m` y uno de los puntos en `y = mx + b` para despejar `b`.  
b) El dominio de una función lineal es siempre `ℝ`. Las intersecciones se hallan evaluando `f(0)` (eje y) y resolviendo `f(x) = 0` (eje x).  
c) Se igualan ambas funciones `f(x) = g(x)` y se resuelve para `x`. Esto equivale a encontrar la solución del sistema formado por ambas ecuaciones, ya que en el punto de intersección ambas comparten las mismas coordenadas `(x, y)`.  
*Respuesta:* a) `m = (−7 − 2) / (−2 − 1) = −9 / −3 = 3`. Sustituyendo en `y = 3x + b` con `(1, 2)`: `2 = 3(1) + b → b = −1`. Función: `f(x) = 3x − 1`.  
b) Dominio: `ℝ`. Intersección eje y: `(0, −1)`. Intersección eje x: `3x − 1 = 0 → x = 1/3`. Punto: `(1/3, 0)`.  
c) `3x − 1 = −x + 4 → 4x = 5 → x = 5/4`. `y = −(5/4) + 4 = 11/4`. Punto de intersección: `(5/4, 11/4)`.

**Ejercicio 17**  
*Procedimiento y justificación:*  
a) Se analiza el denominador. Al contener una raíz cuadrada, el radicando debe ser estrictamente mayor a cero (no puede ser negativo ni cero, ya que está en el denominador). Se resuelve la inecuación.  
b) Se identifican dos puntos de la recta `y = 2x − 1` (interceptos) y se traza la línea.  
c) Se determina el signo de la función. Como el denominador es una raíz cuadrada, siempre es positivo en el dominio. Por lo tanto, el signo de `h(x)` depende únicamente del numerador. Se resuelve la inecuación `2x − 1 > 0` y se intersecta con el dominio.  
*Respuesta:* a) `x + 3 > 0 → x > −3`. Dominio: `(−3, ∞)`.  
b) La recta `y = 2x − 1` corta al eje y en `(0, −1)` y al eje x en `(0.5, 0)`. Se traza una línea recta que pasa por ambos puntos.  
c) Para `h(x) > 0`, se necesita `2x − 1 > 0` (pues el denominador `> 0`). `2x > 1 → x > 0.5`. Intersectando con el dominio `(−3, ∞)`, el intervalo es `(0.5, ∞)`.
